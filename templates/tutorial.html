How to Play

Understand the protocol before attempting decryption.

The system generates a hidden 4-digit sequence. Your objective is to decrypt this sequence using logical guesses.

For every attempt, the system provides structured feedback:

  • Input — The 4-digit number you submitted
  • Present — Digits that exist in the sequence
  • Position — Digits in the correct position

Use previous attempts to eliminate possibilities and progressively synchronize the correct sequence.

The protocol is complete when all four digits are in the correct position.
